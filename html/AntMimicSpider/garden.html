Installing Software

Ninite

There a really cool website that help with the installation of software call Ninite with some awesome features.

    Features

  1. Install and Update All Your Programs at Once.
  2. No toolbars.
  3. No clicking next
  4. Install 64-bit apps on 64-bit machines.
  5. Skip up-to-date apps.
  6. Download apps from each publisher's official site.
  7. Verify digital signatures or hashes before running anything.

    Recommended softwares

  1. Anything other then internet explorer
  2. Malwarebytes.
  3. 7-Zip
  4. Notepad++
  5. Java 8
  6. WinDirStat
  7. Classic Start for Window 8 - 10

Software to avoid

Adobe Flash Player

Drive-by download attacks can happen when Adobe Flash Player is enable on your browser. Learn More

    Problem with Flash Player plugin

  1. Flash Player is dead.
  2. It's buggy.
  3. It crashes a lot.
  4. It requires constant security updates.
  5. It doesn't work on most mobile devices.
  6. Flash makes the web less accessible.
